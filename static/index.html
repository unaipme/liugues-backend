liugues API

still a work in progress¯\_(ツ)_/¯

By using this API, you can access all the information hosted and updated by liugues. Liugues API provides all its information in JSON format.

Below you can find a table with all the developed functionalities, and its URIs. Just make a request to the URIs, with the right parameters and right request type, and you'll receive all the info.

URI Request Response
/g/countries GET

List of all countries from which there are leagues registered in liugues.
/g/leagues GET

List of all leagues that are registered in liugues. Parameters:

  • l_country: Country ID leagues have to match.
/g/teams GET

List of all teams that are registered in liugues.
